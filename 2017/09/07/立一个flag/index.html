这是一个Flag

发表于 2017-09-07 | 分类于 个人随笔

       这是一个Flag。
#在之前的内容会显示在首页，之后的内容会被隐藏，当游客点击Read more才能看到
       已经到了大三了，现在还是浑浑噩噩，没有一件自己特别想去做的事情，感觉日子就这样在我自己的纠结和患得患失中过去了。以前立下的flag也都没有怎么实现，很快就要数模国赛了，忙了一个暑假终于要看到成果了，也希望自己能够挺住，坚持把数模打完。
       已经大三了，确实应该好好考虑一下以后的事情了。高考先不说是个什么情况，到了一个确实没有什么历史氛围和学术氛围的辣鸡211大学是不争的事实，应该为以后做做准备。保研？出国？反正让我找工作是肯定不现实的！身为一个计算机的学生代码能力这么差确实不应该，我现在很后悔大一大二的时候把大量的精力放在了学习上而一度忽视了编程能力的培养，数据结构、算法课是高分又怎么样，只不过是我给自己不打代码找的冠冕堂皇的借口罢了。
       现在看看身边的同学，有的兼顾学习和编程，虽说学的比较累，但是代码能力和学习成绩照样没有落下；那些专心编程的、搞ACM的、学开发的已经参加各种比赛砍瓜切菜、接手项目各种赚钱……人比人气死人，我的基础比他们差的多，他们已经在我前面走了2年的时间，在大学毕业之前超越他们是奢望，但是追寻他们的脚步确实是可以的。
       现在想想以前认真学习还真是可笑哇，目前有去NTU读书的打算，不管怎么样，还是要为梦想拼一把，万一实现了呢？下面是大三上学期的一些计划或者安排：

眼下最重要的就是9.14的数模国赛，国赛对于我和队友来说就是跳板，还是好好弄，不能翻车；
数学竞赛：基本上如果定下来去NTU的话，数学竞赛也就意义不大了，战略放弃吧；
想读NLP，还是需要把英语、统计什么的看看；
争取这学期可以独立参加一次Kaggle；
雅思好好学，英语踏踏实实的看
12月份的PAT：需要提高一下编程能力，争取在国庆节左右把《算法笔记》看完.
